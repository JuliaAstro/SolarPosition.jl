PSA

The PSA (Plataforma Solar de Almería) algorithm is the default high-accuracy solar position algorithm.

SolarPosition.Positioning.PSA — Type
struct PSA <: SolarPosition.Positioning.BasicAlg

PSA (Plataforma Solar de Almería) solar position algorithm.

This algorithm computes solar position with high accuracy using empirical coefficients. Two coefficient sets are available: 2001 (range 1999-2015) and 2020 (range 2020-2050).

Fields

  • coeffs::Int64: Coefficient set year (2001 or 2020)

Constructor

  • PSA(): Uses 2020 coefficients (default)
  • PSA(2020): Uses 2020 coefficients (range 2020-2050, accuracy ±0.0083°)
  • PSA(2001): Uses 2001 coefficients (range 1999-2015, accuracy ±0.0083°)

References

[1] M. Blanco, D. Alarcón, T. López, and M. Lara, "Computing the Solar Vector," Solar Energy, vol. 70, no. 5, 2001, doi:10.1016/S0038-092X(00)00156-0

[2] M. Blanco, K. Milidonis, and A. Bonanos, "Updating the PSA sun position algorithm," Solar Energy, vol. 212, 2020, doi:10.1016/j.solener.2020.10.084

Example

# Use default 2020 coefficients
pos = solar_position(obs, dt, PSA())

# Use 2001 coefficients for historical data
pos_historical = solar_position(obs, dt, PSA(2001))

NOAA

The NOAA (National Oceanic and Atmospheric Administration) algorithm provides an alternative implementation.

SolarPosition.Positioning.NOAA — Type
struct NOAA <: SolarPosition.Positioning.BasicAlg

NOAA (National Oceanic and Atmospheric Administration) solar position algorithm.

This algorithm is based on NOAA's Solar Position Calculator implementation. Currently provides a stub implementation for testing purposes.

Accuracy

Claimed accuracy: ±0.0167° (when fully implemented)

References

[1] NOAA Global Monitoring Laboratory Solar Position Calculator https://gml.noaa.gov/grad/solcalc/calcdetails.html

Status

⚠️ Note: This is currently a stub implementation that returns fixed values. Full implementation is planned for future releases.

Example

# Note: Currently returns stub values
pos = solar_position(obs, dt, NOAA())
